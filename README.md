# dbZero
Writing a database from scratch. Starting with a KV store.

## Why?
We've all used (or heard) a database (relational, noSQL, etc.) at some point but do we really understand how they work underneath? To some people, yes, but I don't think many people do. And I'm one of those people. This is my attempt at understanding databases at the fundamental level.
